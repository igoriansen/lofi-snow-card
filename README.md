# Cozy Lofi Snow Card ❄️🎵

A minimalist, dark-themed interactive landing page designed to deliver a clean visual message. It features a custom canvas-based snowfall simulation, smooth keyframe animations, and automated background audio integration.

## 🚀 Live Demo

You can check out the live project here: [https://igoriansen.github.io/lofi-snow-card/](https://igoriansen.github.io/lofi-snow-card/)

## 🛠️ Technologies Used

* **HTML5:** Semantic structure and interface layout.
* **CSS3:** Custom deep-dark theme, responsive design, and smooth text fade-in transitions.
* **JavaScript (Vanilla):** DOM events, browser audio policy handling, and HTML5 Canvas physics.

## ✨ Key Features

* **Interactive Mechanism:** Engaging "click-to-open" envelope behavior that safely bypasses browser audio restrictions.
* **HTML5 Canvas Snowfall:** A math-driven, lightweight particle simulation that triggers realistic floating snow vectors upon opening.
* **Audio Integration:** Seamless background music loop with custom play/pause state synchronization and audio wave indicators.
* **Fully Responsive:** Clean, centered layout optimized for mobile, tablet, and desktop viewports.
